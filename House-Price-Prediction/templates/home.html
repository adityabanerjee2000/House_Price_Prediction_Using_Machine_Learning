WELCOME TO REAL ESTATE HOUSE PRICE PREDICTION

What is HPP?

House price is closely related to everyone's life, and it is affected by many factors. House price prediction is an important research point for that it can help people to make strategies about house dealing. There are many existing researches about house price prediction. However, existing studies cannot make a comprehensive comparison and analysis of popular housing price forecasting methods. Machine learning and in-depth learning can be used to predict house prices and evaluate the models.

House Price prediction is important to drive Real Estate efficiency. As earlier, House prices were determined by calculating the acquiring and selling price in a locality. Therefore, the House Price prediction model is very essential in the information gap and improving Real Estate efficiency.

Working Procedure To Build Our Model!

Data Collection and Exploration

In our project we are dealing with the dataset which is collected from the Kaggle platform.

Data exploration is examining data to identify patterns and make sense of them in the context of one’s problem.

Data Preprocessing

Data pre-processing is the process of transforming raw data into a form suitable for model development. Data cleaning also falls under data pre-processing. Often we get the dataset, that has irrelevant values or missing values, which needs to be cleaned before fitting into a ML model.

Model Development

Firstly, we spilt our cleaned dataset in such a way that 80% of the known dataset can be used as training set and 20% can be used as test data set. Using the training set ML model is constructed and tested with test data to identify the accuracy level of the model
